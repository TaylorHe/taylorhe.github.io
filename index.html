PROJECTS



Music By You

March 2017
DuckHacks 2017 | Most Marketable

    Music by you is an application that analyzes your emotions and automatically plays a genre related to your mood. First, we take pictures of your face using the computer's webcam in set time intervals. We use Microsoft's Emotion API to analyze emotion, and we parse the json recieved into a vector to be sent to a machine learning algorithm. We had limited data points as we just took a survey, but we trained the neural net to predict which genre of music went along with the emotion. It was created in less than 24 hours by a team of 4 developers, and won a Most Marketable prize at DuckHacks 2017.
    I used Selenium to launch and autonavigate Chrome to Pandora for song searching. In addition, I created the function for picture taking for Mac using OpenCV, and troubleshooted modified driver issues for correct implementation. I created an animated local webpage GUI using HTML/CSS/JS to display current mood and song genre.

View the source code here: Music By You



RockSat

October 2016 - Present

    RockSat is an ongoing project approved by NASA in which I am working with several other engineers and a mechanical engineering professor to place a vibration-free capsule on future NASA payloads. This capsule contains both active and passive vibration isolation. My responsibilities include designing a system to parse the vibrations detected by 8 microphones and actively cancel them using a microprocessor. In addition, I am creating software to post-process accelerometer data after the rocket returns and display them in an easy to read format to verify the success of the isolation chamber.



Text Mining and Stock Analysis

January - May 2017

    This project is about applying natural language processing and machine learning to see how people's opinions on twitter and social media change the stock market. Because of the large influence leaders have, the program will mostly follow political announcements from President Trump. We gathered finance data for Yahoo's stock API. Unfortunately, Twitter's API only gets tweets a week old at most, so instead we are crawling the twitter website with Selenium and scraping tweets. My team and I will be experimenting with different machine learning models to see which provides the most accuracy in prediction. The various components of the project- tweet gathering, stock gathering, tweet sentiment, stock analysis, and front end development - all communicate over a flask server.
